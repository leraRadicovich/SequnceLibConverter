# Инструкция по запуску проекта SequnceLibConverter

## Требования
- Java 17 или выше (у вас установлена Java 21 ✓)
- Maven 3.6+ (или используйте Maven Wrapper)

## Способ 1: Запуск через IDE (Рекомендуется)

### IntelliJ IDEA / Eclipse / VS Code
1. Откройте проект в IDE
2. IDE автоматически определит Maven проект
3. Дождитесь загрузки зависимостей (Maven будет загружать их автоматически)
4. Запустите один из классов:
   - **Swing приложение**: `org.example.SwingEntryPoint` (Desktop GUI)
   - **Веб приложение**: `org.example.Application` (Vaadin Web UI на http://localhost:8080)

## Способ 2: Установка Maven и запуск через командную строку

### Установка Maven

1. Скачайте Maven: https://maven.apache.org/download.cgi
2. Распакуйте в папку (например, `C:\Program Files\Apache\maven`)
3. Добавьте в переменные окружения:
   - `MAVEN_HOME` = `C:\Program Files\Apache\maven`
   - Добавьте в `PATH`: `%MAVEN_HOME%\bin`

### Запуск проекта

#### Сборка проекта:
```bash
cd D:\codeProject\SequnceLibConverter
mvn clean install
```

#### Запуск Swing приложения:
```bash
mvn spring-boot:run -Dspring-boot.run.main-class=org.example.SwingEntryPoint
```

#### Запуск веб-приложения:
```bash
mvn spring-boot:run -Dspring-boot.run.main-class=org.example.Application
```

Или просто:
```bash
mvn spring-boot:run
```
(запустит Application.java по умолчанию)

Веб-интерфейс будет доступен по адресу: http://localhost:8080

## Способ 3: Создание исполняемого JAR

### Сборка JAR файла:
```bash
mvn clean package
```

### Запуск Swing приложения:
```bash
java -jar target/plantuml-ui-0.0.1.jar
```

Примечание: В pom.xml указан mainClass как SwingEntryPoint, поэтому запустится Swing версия.

## Способ 4: Использование Maven Wrapper (если создан)

Если в проекте есть файлы `mvnw.cmd` (Windows) или `mvnw` (Linux/Mac):

```bash
# Windows
.\mvnw.cmd clean install
.\mvnw.cmd spring-boot:run

# Linux/Mac
./mvnw clean install
./mvnw spring-boot:run
```

## Проверка работы

### После запуска Swing приложения:
- Откроется окно "PlantUML Converter"
- Вы сможете выбрать файлы для конвертации/деконвертации

### После запуска веб-приложения:
- Откройте браузер: http://localhost:8080
- Загрузите .puml файл или укажите путь к файлу

## Решение проблем

### Ошибка "Maven not found"
- Установите Maven или используйте IDE (см. Способ 1)
- Или создайте Maven Wrapper

### Ошибка компиляции
- Проверьте версию Java: `java -version` (нужна 17+)
- Очистите проект: `mvn clean`

### Порт 8080 занят
- Измените порт в `application.properties`:
  ```properties
  server.port=8081
  ```

## Структура проекта

- `SwingEntryPoint.java` - Desktop Swing приложение
- `Application.java` - Веб приложение (Spring Boot + Vaadin)
- `MainView.java` - Веб интерфейс (Vaadin)
- Конвертеры находятся в пакетах `converter` и `deconverter`
